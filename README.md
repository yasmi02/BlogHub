# BlogHub
A modern, feature-rich blogging platform built with Django and Bootstrap. Write beautiful posts, follow users, and engage with the community!

#  Features
**Rich Text Editor** - Write posts with formatting buttons (Bold, Italic, Underline, Lists, etc.)
**Multiple Image Support** - Add images anywhere in your posts, not just as cover
**Markdown Support** - Full markdown rendering for beautiful posts
**Tag System** - Organize posts with tags for easy discovery
**Post CRUD** - Create, read, update, and delete your own posts

#  Social Features
**Follow System** - Follow your favorite writers
**Like & Comment** - Engage with posts through likes and comments
**User Profiles** - Customizable profiles with avatar, bio, location, and website
**Follower/Following Stats** - Track your network growth

#  User Experience

**Responsive Design** - Good experience on desktop, tablet, and mobile
**Search Functionality** - Find posts by keywords or tags
**Pagination** - Smooth navigation through posts
**Real-time Notifications** - Success/error messages for all actions

#  Authentication & Security
**User Registration & Login** - Secure authentication system
**Password Hashing** - Secure password storage
**Permission-based Actions** - Users can only edit/delete their own content
